# 📘 C Programming

A collection of C programs covering core computer science concepts — from classic algorithms and data structures to competitive programming solutions and mini-games. Built as a personal learning and reference repository.

---

## 📁 Repository Structure

| File | Description |
|------|-------------|
| `Algorithm.c` | Searching and sorting algorithms with implementations and complexity notes |
| `Data_Structure.c` | Fundamental data structure implementations in C |
| `Linked_List.c` | Singly/doubly linked list operations (insert, delete, traverse) |
| `Bee_Crowd.c` | Solutions to problems from the [Beecrowd](https://judge.beecrowd.com/) online judge |
| `Code_forces.c` | Solutions to problems from [Codeforces](https://codeforces.com/) competitive programming contests |
| `R_P_S_Game.c` | A console-based Rock, Paper, Scissors game |

---

## 🔍 Algorithms Covered (`Algorithm.c`)

### Searching
- **Linear Search** — O(n), scans elements sequentially
- **Binary Search** — O(log n), requires a sorted array

### Sorting
- **Bubble Sort** — simple comparison-based sort
- **Insertion Sort** — builds sorted array one element at a time
- **Selection Sort** — selects the minimum element repeatedly
- **Merge Sort** — divide-and-conquer, O(n log n)
- **Quick Sort** — partition-based, average O(n log n)

## ⚙️ Getting Started

### Prerequisites
- GCC compiler (or any C compiler)
- Linux / macOS / Windows with a terminal

### Compile & Run

```bash
# Clone the repository
git clone https://github.com/khrizve/C_Programming.git
cd C_Programming

# Compile any file
gcc Algorithm.c -o algorithm

# Run
./algorithm
```
